# Collaborative Code Editor

A real-time collaborative code editor built using React, Monaco Editor, Yjs, Socket.IO, Node.js, Express, and Docker.

## Features

- Real-time collaboration
- Active user tracking
- Monaco Editor
- Multi-language support
- Code execution
- Dockerized deployment

## Tech Stack

- React
- Node.js
- Express
- Socket.IO
- Yjs
- Monaco Editor
- Docker

## Run Locally

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
